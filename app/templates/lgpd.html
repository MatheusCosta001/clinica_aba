Política de Privacidade e LGPD

Esta Política descreve como a Clínica ABA realiza o tratamento de dados pessoais no contexto do sistema web utilizado para gerenciamento de atendimentos, prontuários e informações administrativas.

1. Finalidade do Tratamento

Os dados pessoais são coletados para fins clínicos, operacionais e administrativos, incluindo:

  • Cadastro de usuários (pacientes, profissionais, coordenadores e administradores)
  • Organização de atendimentos e sessões clínicas
  • Emissão de relatórios clínicos e financeiros
  • Gestão interna da equipe

2. Dados Coletados

O sistema pode coletar os seguintes dados pessoais:

  • Nome completo
  • E-mail e credenciais de acesso
  • Informações profissionais (papel, especialidade)
  • Dados clínicos inseridos por profissionais autorizados

3. Base Legal

O tratamento ocorre conforme as bases legais da LGPD, especialmente:

  • Execução de contrato (Art. 7º, V)
  • Cumprimento de obrigação legal (Art. 7º, II)
  • Proteção da saúde (Art. 7º, VIII)
  • Legítimo interesse, quando aplicável (Art. 7º, IX)

4. Compartilhamento

Os dados são acessados exclusivamente por profissionais autorizados, respeitando níveis de permissão, tais como:

  • Administrador: gestão geral e registro de usuários
  • Coordenadores: acesso clínico e supervisão
  • Profissionais: registros de atendimentos dos pacientes que atendem

5. Segurança

O sistema adota medidas técnicas para proteção dos dados, como:

  • Criptografia de senhas
  • Controle de acesso por nível de permissão
  • Logs de atividade
  • Armazenamento seguro em banco de dados

6. Direitos do Titular

O titular pode solicitar:

  • Acesso aos seus dados
  • Correção de dados incorretos
  • Anonimização, bloqueio ou eliminação
  • Revogação de consentimento

7. Retenção

Dados clínicos podem ser mantidos pelo período necessário para cumprimento de obrigações legais e para preservação de histórico médico, mesmo após solicitação de exclusão.

8. Contato

Para solicitações relacionadas à LGPD, entre em contato pelo e-mail privacidade@clinicaaba.com.
